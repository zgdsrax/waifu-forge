# Waifu Forge 🦊

An AI-powered web application to generate, render, and customize anime-style (waifu) avatars using stable diffusion models.

## Roadmap
- [ ] Phase 1: Basic AI Generation Backend (Python/FastAPI)
- [ ] Phase 2: Frontend UI for Avatar Generation (React/Vite)
- [ ] Phase 3: Gallery & Storage Integration (S3/Cloudinary)
- [ ] Phase 4: Social Features & Custom LoRA Selection
